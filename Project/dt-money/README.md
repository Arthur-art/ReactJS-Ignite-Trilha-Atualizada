### Trabalhando com ESlint
- Instalar a extensão no vscode
- npm i eslint -D
- npm i @rocketseat/eslint-config -D
- Procurar e corrgir os erros do projeto como um todo
  - npx eslint src --ext .ts,.tsx --fix
